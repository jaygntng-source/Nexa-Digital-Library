## ✨ Tentang NEXA DIGITAL LIBRARY

**NEXA DIGITAL LIBRARY** adalah platform perpustakaan digital modern yang dirancang dengan sentuhan UI/UX premium. Dibangun sepenuhnya dengan HTML, CSS, dan JavaScript murni — tanpa backend, tanpa database, namun tetap powerful dengan penyimpanan lokal.

> 🎯 **Misi**: Menghadirkan pengalaman membaca digital yang elegan, interaktif, dan accessible bagi semua orang.

---

## 🚀 Fitur Unggulan

| Fitur | Deskripsi |
|-------|-----------|
| 📖 **Katalog Buku** | 40+ buku dengan cover, rating, sinopsis, dan informasi lengkap |
| 🔍 **Search & Filter** | Cari judul/penulis, filter kategori, sortir terbaru/populer/rating |
| ❤️ **Favorit System** | Simpan buku favorit dengan localStorage — data tetap ada walau refresh |
| 📖 **Reading Mode** | Mode baca seperti ebook dengan pengaturan font & progress tracker |
| 🧠 **Quiz Pembelajaran** | Uji pemahaman setelah membaca dengan quiz interaktif |
| 🏆 **Gamification** | Dapatkan XP, level, dan badge — Pembaca Pemula, Rajin Membaca, Master Buku, Ilmuwan Muda |
| 👤 **User Profile** | Statistik membaca, XP, level, dan koleksi badge |
| 🛡️ **Admin Panel** | CRUD buku — tambah, edit, hapus dengan mudah |
| 🌗 **Dark/Light Mode** | Tampilan yang nyaman di segala kondisi pencahayaan |
| 📱 **Fully Responsive** | Sempurna di Android, iOS, tablet, laptop, dan desktop |
| ✨ **Animasi Premium** | Glassmorphism, floating elements, micro-interactions, smooth transitions |
